📌 React Todo App

A clean, modern, and beginner-friendly React Todo Application built to help users organize their daily tasks with deadlines, smooth animations, and an intuitive UI.

Note:
This project focuses only on the React frontend.
No backend or database functionality is included — perfect for practicing React concepts like components, props, state management, mapping lists, event handling, animations, and more.

🚀 Features
✅ Add Tasks

Easily add new todo tasks with a title and deadline.

🗂️ Auto-Sorting

Todos are automatically sorted by the closest deadline first, so you always know what’s most urgent.

✨ Animations

Smooth fade-in animations when adding new tasks and visual effects when marking tasks done.

🎯 Mark as Done

Toggle tasks as complete — completed tasks move to the bottom and get visually dimmed with a line-through effect.

🗑️ Delete Tasks

Remove tasks instantly with a delete button.

▶️ How to Run the Project
1. Clone the repo
git clone https://github.com/your-username/react-todo-app.git
cd react-todo-app

2. Install dependencies
npm install

3. Start development server
npm run dev
